Additional Game Modes

Steel Soul Mode

Hardcore game mode unlocked after beating the game the first time in which The Knight does not respawn upon death. Instead, the game ends the save file is reset.


Godseeker Mode

Game mode that takes place exclusively in the Godhome and is intended for fighting the game's boss battles.
